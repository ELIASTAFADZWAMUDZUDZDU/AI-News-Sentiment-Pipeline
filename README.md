# AI News Sentiment Pipeline

## Overview

The AI News Sentiment Pipeline is an end-to-end data engineering and natural language processing project that automatically collects news articles, processes textual data, predicts sentiment using a pre-trained Transformer model, stores the results in PostgreSQL, and visualizes insights through an interactive Streamlit dashboard.

The objective of this project was to demonstrate the integration of modern AI models with data engineering principles by building a complete production-style pipeline.

## Machine Learning Model

The project uses the pre-trained Transformer model:

**distilbert-base-uncased-finetuned-sst-2-english**

This model predicts:

- Positive sentiment
- Negative sentiment

along with a confidence score for every prediction.

Using a pre-trained model allowed the project to leverage transfer learning rather than training a model from scratch.

---

## Database Design

The PostgreSQL database consists of three related tables:

### raw_articles

Stores articles exactly as received from the NewsAPI.

### processed_articles

Stores cleaned article text together with preprocessing metadata.

### sentiment_predictions

Stores predicted sentiment labels and confidence scores generated by the Transformer model.

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-News-Sentiment-Pipeline.git
```

Create a virtual environment

```bash
python -m venv .venv
```

Activate the environment

Windows

```powershell
.\.venv\Scripts\Activate.ps1
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## Running the Project

Run the ETL pipeline

```bash
python pipeline.py
```

Launch the dashboard

```bash
streamlit run app.py
```
